# Black Dragon Coffee House
This is a project created by Asia/Asia B the Techie. 
It is a static website made with plain HTML, CSS, and JavaScript with no use of frameworks or additional libraries.

## Making the Project
This project had base pages, assets, a single stylesheet, and a README file created before uploading to the black-dragon-coffee-house repository on GitHub.
Construction of this project was made possible with the use of the following:
- [W3Schools HTML Tutorial](https://www.w3schools.com/html/default.asp)
- [W3Schools CSS Tutorial](https://www.w3schools.com/css/default.asp)
- [Eric Meyer's CSS Reset](https://meyerweb.com/eric/tools/css/reset/)
